# Setting up CHAT-APP


## Install node_modules in both Client and Server

### Client
 - ### cd client
 - ### `npm install`

### Server
 - ### cd server
 - ### `npm install`

 ---

## Start the APP:

### Client
 - ### cd client
 - ### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Server
 - ### cd server
 - ### `npm install`

Server listens in [http://localhost:4050](http://localhost:4050)
